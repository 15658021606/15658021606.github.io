- 终端输入 vim ~/.bash_profile
- 输入 i 进入编辑模式，然后输入环境变量：

```javascript
export PATH="$M2_HOME/bin:$PATH"
```

- 编辑完成后，按 Esc 退出编辑，输入 :wq! 保存并退出
- 输入 source .bash_profile 使环境变量生效
- 终端输入 echo $PATH 查看当前环境变量，检查添加的变量是否生效
