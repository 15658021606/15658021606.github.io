#### 一、启动和退出

1. 启动

```
vim filename
```

vi 不指定文件名，显示vi的版本信息

![如](/images/vim.png)

vi编辑器有三种模式：命令模式、输入模式、末行模式

* 命令模式：vi 启动后默认进入命令模式，从这个模式可以使用命令切换到另外两种模式， 同时无论在任何模式下只要按[esc]键都可以返回命令模式（注意输入法需要是英文）。命令模式中输入[i]就进入输入模式编辑文件。

* 输入模式：在这个模式中可以进行修改、编辑、输入等工作， 在编辑器最后一行显示“--INSERT--”表示vi进入了输入模式。当我们完成修改输入操作需要保存文件的时候，需要先返回命令模式，再进入末行模式进行保存。

* 末行模式：在命令模式输入“:”，即可进入末行模式。

#### 二、保存和移动

退出

* q! 强制退出，同时没有保存文件, 如果文件已经保存，可以使用q退出，保存退出使用wq,,保存强制退出使用wq!

保存

* w 保存文件， 另存为可以使用 w filename

光标的移动

* 方向键光标移动

翻页命令

  * Ctrl + F 向前翻整页，Ctrl + U 向前翻半页
  * Ctrl + B 向后翻整页，Ctrl + D 向后翻半页

行内快速跳转

* ^ 移动到本行行首
* $ 移动到本行行尾

显示行号和取消行号（末行模式使用）

* set nu 显示行号
* set nonu 取消行号

在命令模式下，可以使用以下命令快速的在行间跳转

* 1G 跳转至行首
* G 跳转至行尾
